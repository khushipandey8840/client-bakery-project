# client-bakery-project# Artisan Bakery

A responsive static HTML shell built using **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Features

* Responsive layout
* Semantic HTML5
* Contact form with validation
* Empty state ("No data found")
* Loading indicator
* Basic XSS input sanitization
* Accessibility support (ARIA labels)
* Analytics console log

## Project Structure

```text
index.html
style.css
script.js
README.md
```

## How to Run

Open `index.html` in any modern web browser.
